# 勤怠管理アプリ



作成した目的　ある企業Atteの勤怠管理システム（人事評価のため）


## 機能一覧

・ログイン機能　　・会員登録機能　　・出勤/退勤/休憩打刻機能　　・日付別確認機能


## 使用技術

Laravel Framework 9.34.0
